# JMCL启动器
正在开发的一个基于Python以及Kivy库的Minecraft启动器。您可以前往[Misaka's Blog](https://blog.geekgame.top "MikuMiku~")查看最新开发进度。您也可以通过Push来参与开发。

# 环境要求

Linux and MacOS:
- Python 3.5+
- kivy

Windows:
- Python 3.5+
- pygame
- docutils 
- pygments 
- pypiwin32 
- kivy.deps.sdl2 
- kivy.deps.glew
- kivy.deps.gstreamer
- kivy.deps.angle
- kivy
